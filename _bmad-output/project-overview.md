# ForkMonkey - Project Overview

## Executive Summary

**ForkMonkey** is a viral GitHub-based digital pet project that combines genetic algorithms, AI evolution, and social mechanics. Users fork the repository to "adopt" a unique monkey that evolves daily using AI (Claude/GPT-4o), with traits inherited through GitHub's fork network.
